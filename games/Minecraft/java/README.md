# Тут егги для JAVA клієнтів гри
---
 
*У папочці /mods знайдете різні мод-лодери, для модових серверів*
*У папочці /plugins знайдете звичайні ядра із підтримкою плагінів*
*У папочці /services знайдете різні дрібні шняги типу Limbo серверів*
*Без папочки валяється чиста офіційне ядро, яке ні модів, ні плагінів не підтримує*

**Рекомендую спершу ОЗНАЙОМИТИСЬ із особливостями кожного ядра**
