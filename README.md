### ❓ Навіщо це існує?
Офіційний репозиторій `pterodactyl/eggs` - це гігантська смітєва яма, де зібрані конфіги для тисячі ігор та скриптів, які вже давно ніхто не використовує. Коли треба швидко розгорнути базову інфраструктуру або підняти ігровий сервер, ти витрачаєш купу часу на пошук потрібного `.json` файлу. 

Тут лежить тільки **база**. Найнеобхідніші та найстабільніші "яйця", відфільтровані від зайвого сміття. Ідеально підходить для швидкого розгортання власних нод.

---

### 🥚 Що лежить у гнізді?
*(Список буде поповнюватись по мірі моєї ліні шукати щось в офіційному репо)*

- **Minecraft:** Найактуальніші ядра (Paper, Purpur, Vanilla, BungeeCord, Velocity).
- **Bots / Apps:** Базові середовища для запуску Python, NodeJS чи Java застосунків.
- **Databases:** Зручні контейнери для Redis, MongoDB тощо.
 
---

### 🛠️ Як імпортувати?

Якщо ти адміниш Птеру, ти і так маєш це знати, але про всяк випадок:

1. Завантажуєш потрібний `.json` файл з цього репозиторію.
2. Заходиш у панель адміністратора Pterodactyl.
3. Переходиш у розділ **Nests** ➡️ Обираєш потрібне гніздо (або створюєш нове).
4. Тиснеш кнопку **Import Egg**.
5. Закидаєш файл і радієш життю.
